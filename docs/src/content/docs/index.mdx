## What it does

`pull-all` walks a directory, finds every git repo, and fast-forwards them all in
parallel — streaming each repo's output into a two-pane TUI while you watch. Drill
into any repo to manage its branches, worktrees, and stashes, and read diffs inline
without leaving the terminal.

<CardGrid stagger>
  <Card title="Parallel pulls" icon="rocket">
    Every repo pulls at once, bounded by a concurrency limit (default: `nproc`).
    Live, per-repo log streaming in a scrollable preview pane.
  </Card>
  <Card title="Branch & worktree management" icon="git-branch">
    A full-screen repo page lists every local branch, worktree, and stash with
    fresh ahead/behind. Check out, fast-forward, delete, or discard — with confirms.
  </Card>
  <Card title="Inline diffs" icon="document">
    Open a 90%-of-screen diff modal on any stash or dirty tree. Toggle between
    uncommitted changes and everything since you forked from the base branch.
  </Card>
  <Card title="Keyboard-first" icon="laptop">
    Vim-style navigation, mouse support, clickable status-bar commands, name + status
    filters, sortable columns, and a tabbed help modal. Jump into
    [lazygit](https://github.com/jesseduffield/lazygit) on any repo with `l`. Works headless
    with `--no-tui`.
  </Card>
</CardGrid>

## Quick start

```bash
# Build and install to ~/bin
make install

# Run in any directory of repos
pull-all ~/projects
```
